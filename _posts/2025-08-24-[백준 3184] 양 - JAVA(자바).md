### 문제

- [[백준] 24445](https://www.acmicpc.net/problem/24445)

![img](/assets/img/algorithm/백준24445.png)
<br /><br />

### 해결방법
- **BFS(너비 우선 탐색)** 활용하기
    - 입력으로 주어진 그래프를 인접 리스트 형태로 저장한다
    - 저장된 노드들을 내림차순으로 정렬한다
    - BFS를 통해 시작 정점(`r`)을 시작으로 각 노드를 방문하면서, 방문 순서를 기록한다

<br /><br />

### 코드

```java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {
    private static int n, m, r;
    private static ArrayList<ArrayList<Integer>> graph = new ArrayList<>();
    private static int[] visited;
    private static StringBuilder sb = new StringBuilder();
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken());
        m = Integer.parseInt(st.nextToken());
        r = Integer.parseInt(st.nextToken());
        visited = new int[n+1];
        
        for (int i = 0; i <= n; i++) {
            graph.add(new ArrayList<>());
        }
        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());
            int u = Integer.parseInt(st.nextToken());
            int v = Integer.parseInt(st.nextToken());

            graph.get(u).add(v);
            graph.get(v).add(u);
        }
        for (int i = 1; i <= n; i++) {
            Collections.sort(graph.get(i), Collections.reverseOrder());
        }

        bfs();
        
        System.out.println(sb);
    }

    private static void bfs() {
        Queue<Integer> q = new LinkedList<>();
        q.add(r);
        visited[r] = 1;
        int idx = 1;

        while (!q.isEmpty()) {
            int node = q.poll();

            for (int i = 0; i < graph.get(node).size(); i++) {
                if (visited[graph.get(node).get(i)] == 0) {
                    idx += 1;
                    q.add(graph.get(node).get(i));
                    visited[graph.get(node).get(i)] = idx;
                }
            }
        }

        for (int i = 1; i <= n; i++) {
            sb.append(visited[i]).append("\n");
        }
    }
}
```
